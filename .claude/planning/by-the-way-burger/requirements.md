# Requirements: BY THE WAY - BURGER & GRILLHAUS

## Business Context
- **Company**: BY THE WAY - BURGER & GRILLHAUS
- **Industry**: Restaurant / Food Service (Burgers, Grill, Balkan Cuisine)
- **Value Proposition**: High-quality burgers and grilled dishes from fresh, locally sourced ingredients. Diverse menu including Smash Burgers, Balkan specialties, and premium options.
- **Target Audience**: B2C - Local residents, tourists, casual diners, families, food enthusiasts in Steinmaur and surrounding areas
- **Geographic Focus**: Switzerland - Steinmaur 8162, Dielsdorf region, greater Zürich area
- **Contact**: Wehntalerstrasse 1, 8162 Steinmaur | 077 290 52 31 | Instagram: BYTHEWAY.CH
- **Service Model**: Dine-in, take-away, free delivery above 15 CHF

## Pages Required
1. **Homepage** (/) - Hero section with brand story, featured menu highlights, customer testimonials, location/hours, Instagram feed
2. **Menu** (/speisekarte) - Complete menu with 9 categories, 38 items, prices in CHF, organized by type (Smash Burgers, Balkan, Grill, etc.)
3. **About Us** (/ueber-uns) - Restaurant story, values, team introduction, sourcing philosophy
4. **Impressum** (/impressum) - Legal information, business registration, contact (noindex, nofollow)
5. **Datenschutz** (/datenschutz) - Privacy policy, GDPR compliance (noindex, nofollow)

## Languages
- **Primary**: German (de) - Single-language site only
- **Secondary**: None

## Features
- Contact information display with phone click-to-call
- Embedded Google Maps for restaurant location
- Opening hours widget (Wed-Fri: 11:00-13:30 & 17:00-21:00, Sat-Sun: 13:00-21:00)
- Customer testimonials carousel
- Full menu with categories, items, descriptions, prices
- Instagram social feed integration
- Mobile-responsive design
- SEO optimization with local schema markup
- Fast page load performance
- Accessibility compliance (WCAG 2.1 AA)
- Analytics integration (Google Analytics 4)
- NO online ordering system (informational only)

- **Local SEO**: Target Steinmaur, Dielsdorf, Bülach, Zürich region

## Technical Requirements
- **Framework**: Next.js 16 (App Router, React Server Components)
- **Styling**: Tailwind CSS + shadcn/ui components
- **Language**: German only (no i18n library needed)
- **Deployment**: Vercel (automatic CI/CD from GitHub)
- **Performance**: Core Web Vitals - LCP <2.5s, FID <100ms, CLS <0.1
- **Browser Support**: Modern browsers (Chrome, Safari, Firefox, Edge - last 2 versions)
- **Mobile-First**: Responsive design, touch-friendly navigation
- **Image Optimization**: Next.js Image component, WebP format, lazy loading
- **SEO**: Metadata API, sitemap.xml, robots.txt, structured data
- **Analytics**: Google Analytics 4, Google Tag Manager
- **Hosting**: Vercel serverless functions, edge network CDN
